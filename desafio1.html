As tags de texto HTML definem textos, estilos de fonte, parágrafos, spans, quebras de linhas. Vamos conhecê-las algumas delas?

Tag "mark":

(voltar)

O Elemento HMTL "mark" representa um trecho de destaque, marca um texto. Utilizado para dar relevância para o usuário, como o resultado de pesquisa.

Exemplo: Não esqueça de comprar leite hoje.

Para marcar a palavra que desejar, utilizar antes da mesma a tag <"mark">

Tag "i":

(voltar)

O elemento HTML "i" pode representar uma parte do texto, na qual a representação tipográfica padrão é no tipo itálico.

A tag "i" é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Exemplo: "A tecnoluogia move o mndo". "Steve Jobs".

Neste caso, para que somente a frase "A tecnologia move o mundo" fique em itálico, basta colocar a tag <"i"> antes da mesma.

Tag "strong":

(voltar)

A tag "strong" é usada para definir o texto com forte importância. O conteúdo dentro é normalmente exibido em negrito . Dica: Use a tag para especificar texto em negrito sem nenhuma importância extra!

Exemplo: A palavra " hoje" , para deixá-la em negrito, basta colocar antes da palavra a tag <"strong">

Hoje é sexta feira

Tag "u":

(voltar)

A tag "u" exibe a palavra em sublinhado. Para isso, a tag deverá ser colocada antes da palavra que deseja sublinhar.

Dica: Evite usar a tag "u" onde possa ser confundido com um hiperlink!

Exmeplo: Vamos sublinhar a palavra "sexta feira" : Hoje é sexta feira

Tag "p":

(voltar)

A tag "p" define um parágrafo. Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada parágrafo. Assim, antes de iniciar cada parágrafo, sempre colocar a tag <"p">
